# Logbook of Venn Diagram Generator

## Introduction
This document is a logbook or journal that records decisions, state changes, discoveries, and key events in reverse chronological order (most recent first). It serves as an immutable project history.

## Entry Format
*   **Timestamp:** `YYYY-MM-DD HH:MM Z`
*   **Author:** Name of person or AI agent.
*   **Entry:** Clear description of the event.

## Entry Categories
*   **State Change** (e.g., task moved to `in_progress`)
*   **Decision** (e.g., technology choice)
*   **Blocker** (creation or resolution)
*   **Discovery** (new findings)
*   **PR Merge**
*   **Exception** (deviation from rules)
